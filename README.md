# ROBOT-FACE
"ESP32 ROBOT WITH FACIAL EXPRESSIONS CHANGE WHEN AN OBJECT DETECTED"
Features 
      - Eye blinking
      - mouth open and close animation
      - object detedection
      - movements (Forward,Backword,Left turn,Right turn,Stop)
Components
      - ESP32
      - OLED display
      - DC Motors
      - H-Bridge motor driver
      - Ultrasonic sensor
Demo Video
      https://github.com/sharmiamuthan/ROBOT-FACE-/blob/main/VIDEO-2026-05-06-00-43-39.mp4
How to run
      1.Upload code to ESP32
      2.Connect OLED,Motors,Sesor,Driver
      3.Power ON
